      • initialize

         final Unit initialize(Context context, String apiKey, String certFilePath, Boolean isDebug, Boolean isLoggingEnabled)

        The first function to be called before calling any other function. This function is used to initialize or prepare the environment that the user is going to connect to.

        Parameters:
        apiKey - provided by Brankas to access the API Services
        certFilePath - directory to fake certificate to be used to bypass SSL
        isDebug - enables the usage of a Sandbox Environment; false for Production, true for Sandbox
        isLoggingEnabled - enables the internal logging mechanism of Brankas to track the statement retrieval flow within the Tap Web App; default value is true

      • checkout

         final Unit checkout(Activity activity, StatementTapRequest tapRequest, CoreListener<String> listener, Integer requestCode, Boolean isAutoConsent, Boolean useRememberMe, String actionBarText, Boolean showBackButton)

        provides the checkout URL to be used to open the Tap Web Application; this function may provide the WebView as well in behalf of the user

        Parameters:
        activity - used to call the internal WebView Activity within the SDK
        tapRequest - contains request to the Tap Web Application
        listener - optional callback to inform successful or failed transaction
        requestCode - Int used as the request code when waiting for the result of the Tap Activity
        isAutoConsent - option to use auto consent function (default value is false)
        useRememberMe - option to use Remember Me function (default value is false)
        actionBarText - optional text in the ActionBar; if null has been passed, the ActionBar will be hidden.
        showBackButton - used to add a back button on the ActionBar; default value is true
